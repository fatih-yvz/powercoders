Positioning

Sticky

To define position of any html element , we use position property It has five different values.

  • static
  • relative
  • absolute
  • fixed
  • sticky

static

it is default value for all html elements. if any element's display value dont set with one of other values we can not arranged top, bottom, right and left properties.

relative

if a html element is set as position:relative, the element are moved with top,bottom,left and right.The important thing is where to take starting point as reference this is major difference between relative and absolute. Relative take next element border for startin point as reference.so we can say relative interact with neigbhoors elements.

absolute

like relative, absolute also allows movement, but there is a difference.Absolute accepts windows's starting point so we can say an absolute element is very free, it can go whereever want.

fixed

an element it's position value are set as fixed can be set with top,bottom,right and left.and it's position dont change wiht scrolling.

Sticky

sticky allows us if we dont scroll the page sticky element stay own place but if we scroll the page and scrolling come to end of the page sticky element stay end of the page.
