Grid Layout

This layout offers us to make easier web design without using positioning and float. we can use row and column like excel

Display Property

An HTML container becomes grid when its display property is set as grid or inline-grid

Gap

we can use gap property to adjust space between rows or columns. First value defines row gap, second value defines column gap.

grid-template-columns

It defines how many column we will have on grid. If we use auto word each column size will be same.

grid-template-row

defines height of each rows

Justify Content

defines where the element is on x axis

align-content

is used for arrange vertical alignment

grid-column

defines where it start and where it end.

grid-row

defines which row line the item will start and end

grid-area property

It is a shorthand for grid-column and grid-row

Naming grid items

can also be used to assign a name to the specified field
